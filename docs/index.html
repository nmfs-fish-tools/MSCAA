Multi-Species Statistical Catch-At-Age

Multi-Species Statistical Catch-At-Age (MSCAA)
Type:
Authors: Klasky, R., Curti, Townsend H., Link, J., and E. Wurst
The Multi-Species Statistical Catch-At-Age tool (MSCAA) allows the user to 1) run a single-species or multi-species catch-at-age model, and 2) visually inspect the resulting Abundance values interactively. The application uses the 3rd party application ADMB with a model (i.e., .tpl and .pin files) supplied by Dr. Kiersten Curti from her 2012 PhD dissertation, Age-Structured Multispecies Model of the Georges Bank Fish Community.
